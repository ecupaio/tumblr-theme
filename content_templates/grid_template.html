Build your site with PureCSS grids:
  • Start with this grid template and add columns with the classes pure-u-1 pure-u-md-*-*
    • Ex) <div class="pure-u-1 pure-u-md-1-3"></div>
  • Add gutters between grid columns by increasing the padding to the $value variable in the _mixins.scss sheet
  • Change grid text and background colors by adding a class to <div class="grid-content"></div> and defining the class in _custom.scss
    • Ex) <div class="grid-content custom-col"></div>
